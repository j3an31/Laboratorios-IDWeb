Algoritmo de búsqueda binaria

  1. Definir los límites de la búsqueda: inicio (el primer elemento) y fin (el último elemento) de la lista.
  2. Calcular el punto medio: medio = (inicio + fin) / 2. Este punto divide la lista en dos mitades.
  3. Comparar el valor buscado:
    • Si el valor en lista[medio] es igual al valor buscado, se ha encontrado y el algoritmo termina.
    • Si el valor buscado es mayor que lista[medio], significa que el elemento solo puede estar en la mitad superior de la lista. Por lo tanto, se ajusta el inicio a (medio + 1).
    • Si el valor buscado es menor que lista[medio], el elemento solo puede estar en la mitad inferior de la lista. Se ajusta el fin a (medio - 1).
  4. Si el bucle termina y el elemento no se ha encontrado, significa que el valor no existe en la lista.

Algoritmo para encontrar el número más grande en una lista

  1. Inicio: Se define una variable llamada "maximo" y se le asigna el primer número de la lista.
  2. Repetición: Se inicia un bucle para recorrer cada número de la lista.
  3. Condición: Dentro del bucle, se compara el número actual con el valor de la variable "maximo". Si el número actual es mayor, se cumple la condición.
  4. Actualización: Si la condición anterior es verdadera, se actualiza el valor de "maximo" con el número actual.
  5. Fin de la repetición: El bucle finaliza cuando se han revisado todos los números de la lista.
  6. Mostrar resultado: Se muestra el valor final de la variable "maximo".
